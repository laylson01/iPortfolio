Esse portfolio foi criado em um template bootstrap, no qual foram alteradas algumas informações e removidas outras. Também foram feitas alterações em alguns detalhes do HTML e CSS. O site está hospedado em um servidor local usando Ngix RP, Portainer, Cloudflare e etc...

O intuito do projeto é voltar praticar HTML, CSS e JS(Bootstrap) e desenvolver novas habilidades voltadas a infraestrutura.
